# 🔩 NetSure MiniUPS Pro — Component Selection

## Overview

NetSure MiniUPS Pro is designed using a modular hardware approach focused on:

* faster development
* realistic prototyping
* compact integration
* practical embedded system design

Instead of building every subsystem from low-level discrete circuitry, the project integrates widely used embedded power modules for improved reliability and scalability.

---

# ⚙️ Core Components

## 🔋 Lithium-Ion Battery System

### Selected Architecture:

* 18650 lithium-ion cells
* modular replaceable battery concept

### Why:

* high energy density
* easy availability
* compact size
* scalable runtime design
* suitable for compact UPS systems

---

# ⚡ USB-C Charging Architecture

### Selected Approach:

* USB-C based charging input

### Why:

* modern charging standard
* improved usability
* compact connector design
* better long-term compatibility
* cleaner product integration

---

# 🔌 IP5306 Power Management Module

### Role:

* charging management
* battery protection
* power routing

### Why Selected:

* compact integrated solution
* simplifies power architecture
* commonly used in portable power systems
* reduces discrete charging complexity
* suitable for modular embedded systems

---

# ⚙️ MT3608 Boost Conversion Modules

### Role:

* voltage step-up conversion
* 9V / 12V output generation

### Why Selected:

* compact module footprint
* adjustable output voltage
* easy integration
* suitable for router backup applications
* simplified prototyping workflow

# 🧠 Design Philosophy

The project prioritizes:

* modularity
* practical engineering
* compact PCB integration
* realistic product architecture
* startup-style hardware design

over unnecessary low-level circuit complexity.

The overall approach focuses on creating a believable product-oriented embedded hardware system rather than a purely academic electronics exercise.
